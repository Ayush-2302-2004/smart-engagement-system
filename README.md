# smart-engagement-system
The Smart Engagement System is an intelligent platform designed to enhance user interaction and engagement through personalized experiences, real-time communication, and data-driven insights.
Technologies like javafx,firebase,postman api are used
• Architected a full-stack community engagement platform integrating a Generative AI chatbot (Google Gemini 
LLM) to autonomously generate dynamic content and moderate discussions. 
• Engineered an intelligent automation engine to create and schedule daily discussion topics, polls, and ice
breakers, resulting in a 45% increase in consistent user interaction. 
• Developed a robust content management backend with advanced search, message pinning, and topic 
categorization, building a searchable knowledge base for the community.
